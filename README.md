## 이 포트폴리오에서 보여주는 것

- AI 기능을 실제 서비스 흐름으로 연결한 경험
- API, 데이터, UI를 하나의 시스템으로 묶어낸 방식
- 역할, 결과 화면, 구조 자료, 발표 자료까지 확인 가능한 프로젝트 기록
