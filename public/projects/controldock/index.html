ControlDock

2025.12 신연준 포트폴리오 프로젝트 상세

여러 Windows 에이전트 PC의 상태 확인, 원격 배포, 공지 브로드캐스트를 통합 관리하는 원격 모니터링 및 배포 시스템입니다.

내 역할

HTTP·TCP·UDP 서버와 백엔드 서비스 로직을 맡아 PC 등록 승인, 상태 체크, 원격 배포 요청 처리, 공지 전송 프로토콜을 구현했습니다.

핵심 포인트

  • 에이전트 등록 및 상태 모니터링
  • 원격 배포 실행·실시간 상태
  • 브로드캐스트 공지 및 수신 추적

기술 스택

  • Python
  • PostgreSQL
  • Docker
  • HTML/CSS/JS
  • TCP/UDP
  • REST API
